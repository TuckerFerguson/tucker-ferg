V.A.P. Density Plot

Interactive iplot of the United State's 2012 Voting-Age Population Density

Data here was obtained from a CSV file containing general election data from the United States 2012 election. To extract the data desired from the .CSV file I utilized Pandas, a python library useful for the handling of large data sets. After obtaining the general election csv data. I used Pandas DataFrame data structure to organize the election data in a way that it could be easily understood and interpreted.

Finally I used a very nifty Python libary called plotly to visualize this data as a iplot. An iplot is a graph that allows for real time interaction of its elements as well as customizations such as axis titles and a legend for the data. It is important to note this plot is opened utillizing a browser, and in order to access an iplot without an active internet connections. Is to the utilize the library cufflinks and its cufflinks.go_offline() method.
