SportEmpire

Sports Empire is the first website I developed from scratch, this was worked on throughout a year long web development course at Boise State University. SportEmpire

Starting with an HTML skeleton, we took an in-depth exploration of what hypertext markup language has to offer. We then utilized a cascading style sheet (.css) to give the DOM elements we had worked with prior a new look and feel. After these basic concepts were covered, we delved into vanilla JavaScript to help give our website a dynamic feel.


We wrapped the course up with some more advanced concepts of back-end development. In order to do so, we utilized PHP as our development language to create a secure login system utilizing validation, sanitization, and encryption methodologies.

Here is a demonstration of solving some logic server side utilizing POST and depending on this logic, reflecting new behavior client side.

In this case registering a new user, having that user log in, and in return starting a session that remembers the current users credentials across several web pages.
