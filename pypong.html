PyPong

Interactive version of the classic "Pong" written in Python

This is made possible utilizing the gui library turtle. Each game object is an turtle element including the canvas in which the game is played.

Event handlers coupled with methods to modify each objects x, and y coordinates enabled the game objects to be dynamic.
